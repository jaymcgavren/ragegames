Game Cheat Devices

Playstation Game Shark

A game cheat device is an add-on you connect to your console system that allows you to mess with a game program through the use of simple codes. Examples would include the Game Genie and Game Shark. In order to use these codes on your console you will need the game cheat device they are designed for. For emulator users, check your documentation to see if your emulator has support for that device.

This section houses RG's lists of codes from amateur authors, many created by me personally. Some are useful, like the stronger multi-directional shot in Zanac. Some are just for fun, like the instant top speed in Sonic the Hedgehog. And some, like the multiplayer code for Final Fantasy Tactics, make an old game into a whole new one...
