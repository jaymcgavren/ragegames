Emulation Utilities

ROM Utilities

Editing Utilities

DOS4GW.EXE Multiple Many emulators require that this executable be in the same directory to run.
Ez-O-Ray Playstation Game Shark users with a Pro Comms Link can use this replacement GS BIOS to upload amateur demos and games to the PS. Easy to use for most applications.
Final Fantasy
Improvement		 NES Graphically edit the overworld map and towns of a Final Fantasy ROM tile-by-tile. Easy to use.
Hexpose Multiple If you find or can make the appropriate reference file for a given ROM, Hexpose will let you easily edit the game's text. A stable program. Also includes a hex editor.
Mario
Improvement		 NES Lets you edit worlds in a Super Mario Bros. ROM. Difficult to figure out, though.
Massage Master
System		 A reasonably fast emulator; doesn't support every game yet though.
Naga SNES A sprite editor that handles multiple bitplanes. The interface could be better but at least it's fast.
Nesticle NES The premiere emulator for NES (or any system, in fact). Supports Net connections for two players in Win95, has a nice GUI, and includes several hacking tools and a debugger right inside.
Nexedit NES A combination hexeditor and tile editor for NES. The graphics editor to use if Nesticle won't work on a ROM.
SearchR Multiple Searches any file (ROM or otherwise) for a text string, even if it's not stored as ASCII. A vital text-editing utility. This is the original, which doesn't have the fancy interface of its sequel, but is much more stable.
UCON Multiple A basic requirement for emulator and backup users. Splits and joins Genesis and SNES ROMS in several formats, applies IPS patches, and more.
X-Late 1.0 Multiple Primitive and difficult to use but probably the best utility to use for translating Japanese ROMs. Make sure to create a file called "XLATE" (no extension) in the same directory or it won't run! X-Late 3.0 is incomplete at this time, sadly.
Zelda
Improvement		 NES The graphical interface lets you edit the overworld of a Legend of Zelda ROM. Easy to figure out but difficult to use.
